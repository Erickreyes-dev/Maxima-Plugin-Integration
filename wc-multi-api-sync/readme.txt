Plugin para sincronizar productos de múltiples proveedores y notificar ventas.

== Description ==
Permite registrar proveedores externos, mapear JSON a productos de WooCommerce, sincronizar por Action Scheduler y notificar ventas completadas.

== Installation ==
1. Copia la carpeta `wc-multi-api-sync` a `wp-content/plugins/`.
2. Activa el plugin.
3. Configura proveedores y mapeos en el menú WC Multi API Sync.

== Frequently Asked Questions ==
= ¿Cómo probar el endpoint? =
Usa el botón "Test Endpoint" en la página de mapeos.
